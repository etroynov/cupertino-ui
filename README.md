## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app in the development mode.<br>
Open [http://localhost:8080](http://localhost:8080) to view it in the browser.

The page will reload if you make edits.<br>
You will also see any lint errors in the console.

### `npm run dev`

Builds the app for production to the `build` folder.<br>
It correctly bundles React in production mode without optimization.

### `npm run build`

Builds the app for production to the `build` folder.<br>
It correctly bundles React in production mode and optimizes the build for the best performance.

The build is minified and the filenames include the hashes.<br>
Your app is ready to be deployed!

### `npm run storybook`

Storybook should start, on a random open port in dev-mode.<br>
Now you can develop your components and write stories and see the changes in Storybook immediately since it uses Webpackâ€™s hot module reloading.
